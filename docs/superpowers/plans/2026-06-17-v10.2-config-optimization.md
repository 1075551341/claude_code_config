# 时间处理模板（禁止 new Date() / datetime.now()）

> 铁律: 业务逻辑禁止裸调原生时间 API，必须通过 Clock 接口依赖注入。

## Python (pendulum)

```python
import pendulum
from abc import ABC, abstractmethod

class Clock(ABC):
    @abstractmethod
    def now(self) -> pendulum.DateTime: ...

class SystemClock(Clock):
    def now(self) -> pendulum.DateTime:
        return pendulum.now()

# 业务代码
def is_expired(deadline: pendulum.DateTime, clock: Clock) -> bool:
    return clock.now() > deadline
```

## TypeScript (dayjs)

```typescript
import dayjs, { Dayjs } from 'dayjs';

interface Clock {
  now(): Dayjs;
}

class SystemClock implements Clock {
  now(): Dayjs { return dayjs(); }
}
```
```

# 错误处理模板（R16: 禁止裸 except:pass）

## Python

```python
class AppError(Exception):
    """业务异常基类，携带机器可读 code"""
    def __init__(self, code: str, message: str, detail: dict | None = None):
        self.code = code
        self.message = message
        self.detail = detail or {}
        super().__init__(message)

# 已知错误 → 业务码 + 友好提示
try:
    result = await db.query(...)
except DatabaseError as e:
    raise AppError("DB_001", "数据库查询失败", {"original": str(e)}) from e

# 未知错误 → 完整日志 + 通用错误码
try:
    result = risky_operation()
except Exception as e:
    logger.exception("risky_operation failed", extra={"input": input_summary})
    raise AppError("INTERNAL", "操作失败，请稍后重试") from e
```

# 不可变操作（map/filter/reduce）

> CORE.md: "不可变优先：创建新对象，数组用展开/map/filter，不原地修改"

## 数组

```typescript
// ✅ 不可变
const active = users.filter(u => u.active);
const names = users.map(u => u.name);
const updated = users.map(u => u.id === id ? { ...u, name: newName } : u);

// ❌ 原地修改
users.push(newUser);           // 禁止
users.sort((a, b) => ...);     // 用 [...users].sort() 代替
```

## 对象

```typescript
// ✅ 展开创建新对象
const updated = { ...config, debug: true };

// ✅ 深层更新
const state = { ...prev, user: { ...prev.user, name: newName } };

// ❌ 原地修改
config.debug = true;           // 禁止
delete obj.key;                // 用解构代替

# 异步模式（禁止裸 await）

> CORE.md: "异步操作必须 try/catch，禁止裸 await"

## 标准 async/await

```typescript
async function fetchUser(id: string): Promise<User> {
  try {
    const response = await api.get(`/users/${id}`);
    return response.data;
  } catch (error) {
    throw new AppError('API_001', `获取用户失败: ${id}`, { original: error });
  }
}
```

## Promise.all 并发

```typescript
const [users, posts] = await Promise.all([
  fetchUsers().catch(e => { logger.error('users failed', e); return [] as User[]; }),
  fetchPosts().catch(e => { logger.error('posts failed', e); return [] as Post[]; }),
]);
```

## 防抖/节流

```typescript
function debounce<T extends (...args: any[]) => any>(fn: T, ms: number) {
  let timer: ReturnType<typeof setTimeout>;
  return (...args: Parameters<T>) => {
    clearTimeout(timer);
    timer = setTimeout(() => fn(...args), ms);
  };
}
```
