# 办公文档处理

## 文档类型速查

| 类型 | 工具/技能 | 要点 |
|------|----------|------|
| Excel | xlsx skill | 公式、图表、数据透视、条件格式 |
| Word | docx skill | 样式模板、修订追踪、目录生成 |
| PPT | pptx skill | 母版设计、动画、演讲者备注 |
| PDF | pdf skill | 提取/合并/水印/表单填写/OCR |
| 邮件 | business-writing agent | 语气适配、结构化、附件提醒 |
| 会议纪要 | business-writing agent | 决议/待办/负责人/截止日期 |
| 合同 | 逐条审查+风险标注 | 义务/违约/期限/管辖权 |
| 通知公告 | 简洁+完整+无歧义 | 5W1H(谁/何事/何时/何地/为何/如何) |

## 写作原则

1. **先问后写**：确认受众、目的、语气、长度要求再动笔
2. **结构先行**：先列大纲再填充，保证逻辑连贯
3. **一页原则**：报告/PPT核心信息一页可读，细节放附录
4. **行动导向**：每段结尾指向具体行动(谁在何时做什么)
