# /archive — 归档已完成规格

将已完成的规格提案归档到 `openspec/changes/archive/`。

## 流程

1. 确认所有 tasks 已完成、验证通过
2. 移动 `openspec/changes/CHANGE_NAME/` → `openspec/changes/archive/DATE-CHANGE_NAME/`
3. 更新相关规格文档（如有主规格文件）
4. 提交归档记录

## 输出

```
归档完成: openspec/changes/archive/2026-04-24-add-dark-mode/
更新文件: 3 个规格文档已更新
```
