# 头脑风暴（强制执行）

> **铁律：稍复杂任务必须先发散再收敛，禁止直接开干。**

## 强制流程

```
稍复杂任务
├─ 阶段1: 发散（必须）→ 列出≥3种方案
├─ 阶段2: 评估（必须）→ 六帽思考/决策矩阵
├─ 阶段3: 收敛（必须）→ 选最优方案+说明理由
└─ 阶段4: 确认（必须）→ 与用户确认后执行
```

## 发散方法（按需选用）

### 六顶思考帽（方案评估必用）

| 帽色 | 视角 | 必问 |
|------|------|------|
| 白 | 事实 | 已知什么？缺什么信息？ |
| 红 | 直觉 | 感觉如何？最担心什么？ |
| 黑 | 风险 | 可能出什么问题？为什么失败？ |
| 黄 | 价值 | 好处是什么？为什么值得做？ |
| 绿 | 创意 | 还有什么可能？如何改进？ |
| 蓝 | 流程 | 下一步做什么？需要什么决策？ |

### SCAMPER（创新时使用）

S替代 / C组合 / A调整 / M修改 / P其他用途 / E消除 / R反转

### 5W1H（需求分析必用）

What(做什么) / Why(为什么) / Who(谁做/为谁) / When(何时) / Where(何地) / How(如何)

## 评估框架（方案对比必用）

### 决策矩阵

```
方案 | 成本 | 时间 | 风险 | 收益 | 总分
-----|------|------|------|------|-----
A    |  X   |  X   |  X   |  X   |  X
B    |  X   |  X   |  X   |  X   |  X
```

### RICE优先级

分数 = (Reach × Impact × Confidence) / Effort

## 输出格式

```markdown
## 方案对比
[决策矩阵]

## 推荐方案: [名称]
理由: [1-3条]

## 备选方案: [名称]
适用条件: [何时选此方案]
```
