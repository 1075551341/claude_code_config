# 进度追踪

## 触发条件
- 执行长任务（>3步骤）
- 用户要求追踪进度
- 子Agent执行多步骤任务

## 执行步骤

1. **初始化进度**：在任务开始时创建 checkbox 列表
   ```
   - [ ] 步骤1：描述
   - [ ] 步骤2：描述
   - [ ] 步骤3：描述
   ```

2. **动态更新**：每完成一个步骤，更新对应 checkbox
   ```
   - [x] 步骤1：描述 ✓
   - [ ] 步骤2：描述（进行中...）
   - [ ] 步骤3：描述
   ```

3. **完成标记**：所有步骤完成后输出总结

## 验收标准
- 每个步骤有对应 checkbox
- 完成步骤及时更新
- 最终所有 checkbox 标记为完成
