## 时间管理方法

### 番茄工作法

```
1. 选择任务
2. 设定25分钟计时
3. 专注工作
4. 休息5分钟
5. 每4个番茄休息15-30分钟

工具推荐:
- Forest
- Tomato Timer
- Pomodoro Timer
```

### GTD (Getting Things Done)

```
5个步骤:
1. 收集: 记录所有事项
2. 理清: 判断是否可执行
3. 整理: 分类归档
4. 回顾: 定期检查
5. 执行: 按情境行动

分类:
- 下一步行动
- 等待清单
- 将来/也许
- 参考资料
- 项目清单
```

### 艾森豪威尔矩阵

```
        紧急          不紧急
重要  | 立即执行    | 计划安排
      | 第一象限    | 第二象限
------+-------------+-------------
不重要| 授权/委托   | 减少消除
      | 第三象限    | 第四象限

优先处理: 重要不紧急(第二象限)
```

### 时间块

```
早上: 深度工作 (9:00-12:00)
下午: 会议/沟通 (14:00-17:00)
晚上: 学习/阅读 (19:00-21:00)

原则:
- 同类任务批量处理
- 保护深度工作时间
- 预留缓冲时间
```

## 效率技巧

### 消除干扰

```
1. 关闭无关通知
2. 设置专注模式
3. 环境整理
4. 告知他人勿扰
```

### 批量处理

```
- 邮件集中回复 (固定时段)
- 会议集中安排
- 同类任务批量完成
```

### 二分钟法则

```
如果一件事能在2分钟内完成，立即执行
不要添加到待办列表
```

### 精力管理

```
高精力时段: 深度工作、重要决策
中精力时段: 常规工作、沟通协作
低精力时段: 整理归档、简单任务
```

## 注意事项

```
必须:
- 保护深度工作时间
- 预留缓冲时间
- 定期回顾调整
- 记录时间消耗

避免:
- 过度规划
- 忽略休息
- 频繁切换任务
- 忽视精力管理
```
