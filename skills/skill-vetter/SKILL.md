# 技能安全审查

## 审查项目

- 代码安全性检查
- 恶意代码检测
- 权限范围验证
- 依赖安全扫描
- 隐私泄露风险

## 审查流程

```
代码分析 → 静态扫描 → 动态测试 → 风险评估 → 生成报告
```

## 安全标准

- OWASP Top 10
- 依赖漏洞扫描
- 权限最小化原则
- 数据加密要求

## 使用场景

- 新技能安装前
- 技能更新验证
- 定期安全审计
- 合规性检查
