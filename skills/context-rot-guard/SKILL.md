# 上下文腐烂守卫

## 问题
随着对话上下文窗口填充，AI输出质量退化（context rot）。

## 监控
- 上下文使用率 >70%：触发压缩提醒
- 上下文使用率 >90%：强制压缩

## 压缩策略
1. 摘要已完成子目标的上下文
2. 保留关键决策点和验证证据
3. 丢弃重复和已验证的中间步骤
4. 必要时启动新子Agent（fresh context window）

## 长任务治理
- 每完成一个子目标：输出状态摘要
- 长任务（>30分钟）：拆分为独立子Agent
- 每个子Agent有明确成功标准
