# Changelog

All notable changes to this agent skill will be documented in this file.

## [1.0]  -2026-03-19

### Added
- 初始版本的极端边界测试场景分析技能
- 支持深度扫描业务代码的分支路径，识别并推荐开发人员容易遗漏的极端边界测试场景
- 支持 Java、Python、JavaScript、Go 等主流代码语言的分析
- 提供结构化的JSON格式测试报告输出
- 涵盖9个核心测试维度：数据边界、时间边界、配置边界、序列化边界、逻辑组合、事件机制、依赖管理、并发安全、资源释放

### Changed
- 优化了JSON格式验证要求，确保所有数字值完整且特殊字符正确转义
- 完善了置信度(confidence)的数值范围限制(0.0-1.0)和严重程度(severity)的分类
- 统一了测试报告的输出结构和数据处理规则

### Features
- 自动识别潜在边界测试场景并生成相关测试用例
- 提供高置信度的边界条件分析和测试策略建议
- 支持参数化测试和边界测试方法推荐
- 集成白盒测试原则，确保测试用例的质量和覆盖率

## [1.0] - 2026-03-19

### Added
- 技能概念设计和核心功能规划
- 基础的边界测试场景分析方法论制定
- 测试维度分类和典型事例收集
- JSON输出格式规范设计
- 使用指南和最佳实践框架制定
