## Twitter/X 优化

### 算法因素

```
影响可见度的因素：
- 互动率（回复、转发、点赞）
- 停留时间（阅读时长）
- 媒体类型（图片、视频、GIF）
- 发布时间
- 内容新鲜度
- 用户关系强度
```

### 最佳实践

```
内容优化：
- 开头吸引注意（前2行）
- 使用简洁语言
- 包含行动号召
- 适度使用话题标签（1-2个）
- 添加视觉元素

格式建议：
- 长度：100-280字符最佳
- 段落：1-4段
- 添加相关链接
```

### 发布时间

```
最佳发布时段（UTC）：
- 早间：7-9am
- 午间：12-1pm
- 晚间：5-7pm

根据目标受众时区调整
```

## LinkedIn 优化

### 内容类型

```
高表现类型：
- 专业见解/分析
- 行业趋势评论
- 个人成长故事
- 实用技巧分享
- 数据可视化
```

### 格式建议

```
- 开头提出问题或观点
- 使用段落分隔（2-4段）
- 包含具体数据或案例
- 结尾提出问题引发讨论
- 添加相关话题标签
```

## 数据分析

### 关键指标

| 指标 | 计算 | 目标 |
|------|------|------|
| 互动率 | (回复+转发+点赞)/曝光 | > 3% |
| 点击率 | 点击/曝光 | > 1% |
| 增长率 | 新关注/时段 | 稳定上升 |
| 曝光增长 | 总曝光/时段 | 持续增加 |

### 分析周期

```
短周期：每日监控关键指标
中周期：每周分析内容表现
长周期：每月评估策略效果
```

## 注意事项

```
必须：
- 保持真实声音
- 提供真实价值

- 定期分析数据

避免：
- 过度推广
- 话题标签滥用
- 自动化过度
- 忽视互动回复
```
