## 学习路径

### 前端开发

```markdown
## 初级 (0-6个月)
1. HTML/CSS 基础
2. JavaScript 基础
3. Git 版本控制
4. 响应式布局

## 中级 (6-18个月)
1. ES6+ 高级语法
2. React/Vue 框架
3. TypeScript
4. 构建工具 (Webpack/Vite)

## 高级 (18个月+)
1. 性能优化
2. 架构设计
3. 微前端
4. Node.js 后端
```

### 后端开发

```markdown
## 初级 (0-6个月)
1. 编程语言基础 (Python/Go/Java)
2. 数据库基础 (SQL)
3. HTTP 协议
4. Git 版本控制

## 中级 (6-18个月)
1. Web 框架 (FastAPI/Spring/Gin)
2. 数据库优化
3. 缓存 (Redis)
4. API 设计

## 高级 (18个月+)
1. 微服务架构
2. 分布式系统
3. 消息队列
4. 容器化 (Docker/K8s)
```

### 全栈开发

```markdown
## 必备技能
- 前端框架 (React/Vue)
- 后端语言 (Node.js/Python/Go)
- 数据库 (SQL + NoSQL)
- DevOps 基础

## 推荐路线
1. 先精通一侧 (前端或后端)
2. 学习另一侧基础
3. 全栈项目实践
4. 架构能力提升
```

## 学习方法

### 费曼学习法

```
1. 选择一个概念
2. 用简单语言解释
3. 发现知识漏洞
4. 回顾并简化
```

### 刻意练习

```
1. 明确目标
2. 专注练习
3. 即时反馈
4. 纠正改进
```

### 项目驱动学习

```
1. 确定项目目标
2. 拆解技术需求
3. 边学边做
4. 总结复盘
```
