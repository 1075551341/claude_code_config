# 简历生成器

## 核心能力

**根据职位描述定制简历、突出匹配技能、优化简历内容。**

---

## 适用场景

- 定制求职简历
- 简历内容优化
- 职位匹配分析
- 简历格式调整

---

## 简历结构

### 标准模板

```markdown
# 姓名
职位目标：[目标职位]

## 联系方式
- 邮箱：xxx@email.com
- 电话：xxx-xxxx-xxxx
- GitHub：github.com/xxx
- LinkedIn：linkedin.com/in/xxx

## 专业技能
- 技能类别1：技能A、技能B、技能C
- 技能类别2：技能D、技能E、技能F

## 工作经历
### 公司名称 | 职位 | 时间
- 主要职责描述
- 关键成就（量化）

## 项目经验
### 项目名称 | 角色 | 时间
- 项目描述
- 技术栈
- 主要贡献

## 教育背景
### 学校 | 专业 | 学位 | 时间
```

## 内容优化

### 职位匹配分析

```markdown
## 分析流程

1. 提取职位要求关键词
2. 对比自身技能
3. 突出匹配项
4. 弱化不相关项
5. 补充缺失技能（如正在学习）
```

### 技能匹配表

| 职位要求 | 我的匹配度 | 简历体现 |
|----------|-----------|----------|
| React开发 | ★★★★★ | 工作经历、项目经验 |
| TypeScript | ★★★★☆ | 技能列表、项目描述 |
| Node.js | ★★★☆☆ | 项目经验 |

### 成就量化

```
避免：
- "负责系统开发"
- "提高了性能"
- "参与了项目"

推荐：
- "主导开发XX系统，服务10万+用户"
- "优化查询性能，响应时间降低60%"
- "作为核心成员参与XX项目，按时交付"
```

## 简历定制流程

### 1. 分析职位描述

```markdown
提取关键词：
- 必备技能：React, TypeScript, Node.js
- 加分技能：GraphQL, Docker, AWS
- 经验要求：3年以上
- 行业背景：金融科技优先
```

### 2. 调整简历内容

```markdown
根据匹配度调整：
- 重新排序技能列表（匹配高的在前）
- 突出相关项目经验
- 添加相关关键词
- 调整工作描述侧重点
```

### 3. 输出定制简历

```markdown
## 输出格式

### 基本信息
[根据申请调整]

### 专业技能（按匹配度排序）
1. [最匹配技能]：详细描述
2. [次匹配技能]：简要描述

### 相关经验（突出相关项目）
[调整描述角度]

### 其他信息
[补充匹配关键词]
```

## 不同职位模板

### 技术岗位

```markdown
## 专业技能
**前端**：React, Vue, TypeScript, Webpack
**后端**：Node.js, Python, PostgreSQL
**工具**：Git, Docker, CI/CD

## 项目经验
### 电商平台 | 前端负责人 | 2023.01-至今
- 技术栈：React + TypeScript + Redux
- 负责核心模块开发
- 性能优化：首屏加载从3s降至1.2s
```

### 产品岗位

```markdown
## 专业技能
**产品**：需求分析、原型设计、数据分析
**工具**：Figma, Axure, SQL, Tableau

## 项目经验
### 用户增长项目 | 产品经理 | 2023.01-至今
- 设计增长策略，月活提升30%
- 主导A/B测试，优化转化率
```

## 注意事项

```
必须：
- 针对职位定制
- 量化工作成果
- 使用关键词
- 控制在1-2页

避免：
- 一份简历投所有
- 虚假信息
- 过于冗长
- 格式混乱
```
