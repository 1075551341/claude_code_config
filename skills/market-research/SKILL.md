## 调研框架

### PEST分析

```markdown
政治因素(Political)：
- 政策法规变化
- 行业监管要求
- 税收政策影响
- 国际贸易关系

经济因素(Economic)：
- GDP增长趋势
- 消费者购买力
- 利率汇率变化
- 通货膨胀率

社会因素(Social)：
- 人口结构变化
- 消费习惯演变
- 文化价值趋势
- 生活方式改变

技术因素(Technological)：
- 新技术突破
- 研发投入趋势
- 技术替代威胁
- 数字化程度
```

### SWOT分析

```markdown
优势(Strengths)：
- 核心竞争力是什么
- 有哪些独特资源
- 品牌/技术/渠道优势

劣势(Weaknesses)：
- 资源短板在哪里
- 能力缺口是什么
- 成本/效率劣势

机会(Opportunities)：
- 市场增长空间
- 政策红利
- 技术变革机会
- 竞争对手失误

威胁(Threats)：
- 新进入者威胁
- 替代品风险
- 行业衰退风险
- 成本上涨压力
```

## 行业研究

### 市场规模测算

```python
# TAM/SAM/SOM模型
def calculate_market_size():
    """
    TAM - Total Addressable Market（总可触达市场）
    SAM - Serviceable Addressable Market（可服务市场）
    SOM - Serviceable Obtainable Market（可获得市场）
    """
    # 自上而下法
    tam = total_market_value  # 行业总规模
    sam = tam * serviceable_ratio  # 可服务比例
    som = sam * obtainable_ratio  # 可获得比例

    # 自下而上法
    target_users = potential_user_count
    usage_frequency = times_per_year
    price_per_use = average_price
    som = target_users * usage_frequency * price_per_use

    return {
        'TAM': tam,
        'SAM': sam,
        'SOM': som
    }
```

### 行业生命周期判断

```markdown
导入期特征：
- 用户认知度低
- 技术不成熟
- 市场规模小
- 竞争者少

成长期特征：
- 用户快速增长
- 技术逐渐成熟
- 市场规模扩大
- 竞争者增多

成熟期特征：
- 用户增长放缓
- 技术标准化
- 市场饱和
- 价格竞争激烈

衰退期特征：
- 用户流失
- 替代品出现
- 市场萎缩
- 企业退出
```

## 用户调研

### 调研方法

```markdown
定性研究：
- 用户访谈（1对1深度访谈）
- 焦点小组（6-8人小组讨论）
- 用户观察（现场行为观察）
- 日记研究（长期行为记录）

定量研究：
- 问卷调查（大规模数据收集）
- A/B测试（方案对比验证）
- 数据分析（行为数据挖掘）
- 眼动追踪（视觉注意力分析）
```

```markdown
# 用户画像

## 基本信息
```
