### 批量重命名

```python
import os
import re
from datetime import datetime

def rename_invoices(folder_path):
    """
    批量重命名发票文件

    规则：提取日期、金额，生成规范文件名
    """
    for filename in os.listdir(folder_path):
        if not filename.endswith(('.pdf', '.jpg', '.png')):
            continue

        # 从OCR结果或文件名提取信息
        # 这里是示例逻辑
        old_path = os.path.join(folder_path, filename)

        # 新文件名
        new_name = f"2025-01-15_办公_358.00_发票.pdf"
        new_path = os.path.join(folder_path, new_name)

        os.rename(old_path, new_path)
        print(f"Renamed: {filename} -> {new_name}")
```

### 年度汇总

```python
def generate_annual_summary(invoices):
    """
    生成年度发票汇总

    参数：
        invoices: list of dict [{date, type, amount, tax}]

    返回：
        dict: 年度汇总数据
    """
    summary = {}

    for inv in invoices:
        month = inv['date'][:7]  # YYYY-MM
        category = inv['type']

        if month not in summary:
            summary[month] = {}
        if category not in summary[month]:
            summary[month][category] = {'count': 0, 'amount': 0, 'tax': 0}

        summary[month][category]['count'] += 1
        summary[month][category]['amount'] += inv['amount']
        summary[month][category]['tax'] += inv.get('tax', 0)

    # 计算年度总计
    annual_total = {
        'total_count': sum(
            summary[m][c]['count']
            for m in summary for c in summary[m]
        ),
        'total_amount': sum(
            summary[m][c]['amount']
            for m in summary for c in summary[m]
        ),
        'total_tax': sum(
            summary[m][c]['tax']
            for m in summary for c in summary[m]
        )
    }

    return {
        'monthly': summary,
        'annual': annual_total
    }
```

## 税务准备

### 进项税抵扣

```markdown
可抵扣发票类型：
- 增值税专用发票
- 通行费发票
- 农产品收购发票
- 海关进口增值税专用缴款书

不可抵扣类型：
- 增值税普通发票
- 定额发票
- 用于免税项目的进项
- 用于集体福利的进项

抵扣期限：
- 专用发票：开具之日起360日内
- 通行费：开具之日起360日内
```

### 发票查验

```markdown
查验渠道：
- 国家税务总局全国增值税发票查验平台
- 各省市税务局网站
- 手机APP（如"发票查验"）

查验要点：
1. 发票代码+号码
2. 开票日期
3. 校验码后6位
4. 金额

异常情况处理：
- 发票信息不一致：联系开票方
- 发票作废：不得报销
- 红字发票：注意冲销关系
```
