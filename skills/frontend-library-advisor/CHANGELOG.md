# 变更日志

本文档记录了前端包智能推荐技能（frontend-library-advisor）的所有重要更改。

## [1.0.2] - 2026-04-07

### 改进

- **输出格式优化**: 修复偶现输出内容包含了 markdown 源代码问题

## [1.0.1] - 2026-03-25

### 新增

- **严格核心工作流程**：五步法执行流程（需求理解、候选包生成、评估筛选、包存在性验证、精准推荐）
- **智能触发检测机制**：自动检测用户输入中的触发词，支持基础触发词、技术触发词、功能触发词和问题触发词
- **深度语义分析**：增强的关键词提取，支持层次化关键词提取（核心关键词 + 相关同义词）
- **框架智能识别**：自动识别前端框架偏好（React/Vue/Angular 等）
- **智能候选包生成**：基于 AI 知识库直接生成 5-10 个相关候选包，避免外部脚本搜索依赖
- **智能评估系统**：功能相关性分析、框架兼容性评估、社区活跃度判断、使用流行度评估、技术成熟度评级

### 改进

- **输出格式优化**：推荐包必须包含包名称、版本、评分、安装指南、使用示例和备选方案建议
- **触发条件完善**：支持隐含需求的自动触发，技术讨论中主动触发
- **验证流程优化**：多级验证机制确保数据准确性和推荐可靠性
- **支持场景扩展**：增加了项目初始化时的技术栈建议、代码重构时的库升级建议等场景

## [1.0.0] - 2026-03-17

### 新增

- **核心功能**：基于多维度评估的前端包智能推荐系统
- **智能搜索**：多轮搜索策略，支持核心关键词和扩展关键词组合查询
- **科学评估**：基于下载量、活跃度、依赖健康度的多维度评分算法
- **框架支持**：全面支持 React/Vue/Angular 等前端框架
- **包验证**：多级验证机制确保包存在性和兼容性
- **精准推荐**：返回综合评分最高的单一最佳包推荐

### 使用场景

- 为特定功能寻找最佳 npm 包
- 性能优化和包替换建议
- 安全性检查和包健康度评估
- 多框架环境下的包选择决策
- 包依赖管理优化和瘦身
