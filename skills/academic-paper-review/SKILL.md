# 学术论文结构化审查

## 审查维度（顶会标准）

### 1. 摘要与贡献

- [ ] 摘要是否准确反映论文内容？
- [ ] 核心贡献是否清晰列出？
- [ ] 贡献是否具有新颖性（vs 增量改进）？

### 2. 方法论

- [ ] 方法描述是否足够复现？
- [ ] 理论基础是否扎实？
- [ ] 假设是否明确且合理？
- [ ] 与现有方法的关系是否清楚？

### 3. 实验评估

- [ ] 实验设计是否公平？
- [ ] 基线方法是否充分？
- [ ] 评估指标是否合适？
- [ ] 统计显著性是否报告？
- [ ] 消融实验是否完整？

### 4. 写作质量

- [ ] 逻辑是否连贯？
- [ ] 图表是否清晰且必要？
- [ ] 引用是否完整且准确？
- [ ] 语言是否精确无歧义？

### 5. 局限性与未来工作

- [ ] 局限性是否诚实讨论？
- [ ] 未来方向是否合理？
- [ ] 潜在负面影响是否考虑？

## 评分标准

| 维度       | 权重 | 评分(1-5) |
| ---------- | ---- | --------- |
| 新颖性     | 30%  | -         |
| 技术质量   | 25%  | -         |
| 实验充分性 | 25%  | -         |
| 清晰度     | 10%  | -         |
| 影响力     | 10%  | -         |

## 输出格式

```markdown
# 论文审查：[标题]

## 总评：Strong Accept / Accept / Weak Accept / Borderline / Weak Reject / Reject

## 各维度评分

[评分表]

## 优势

1. [具体优势]

## 不足

1. [具体问题 + 改进建议]

## 关键问题（必须回答）

1. [对论文的致命问题提问]
```
